# ⌨️ Auto Typer

**Auto Typer** is a simple Python automation script designed for repetitive text typing and message sending.
It helps automate high-volume typing tasks, making it useful for testing, productivity, and bulk messaging workflows.

## ✨ Features

| Feature                 | Description                                     |
| ----------------------- | ----------------------------------------------- |
| ⚡ Auto Typing           | Automatically types text multiple times         |
| 🔁 Repetition Control   | Set the number of times messages should be sent |
| ⏳ Delay Support         | Add custom delay between each message           |
| 🖱️ Keyboard Automation | Uses PyAutoGUI for seamless keyboard actions    |
| 🛠️ Lightweight Script  | Simple and easy-to-run Python script            |
| 💻 Productivity Tool    | Useful for testing and repetitive tasks         |

| Technology       | Purpose                       |
| ---------------- | ----------------------------- |
| 🐍 **Python**    | Core programming language     |

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/KrishBharadwaj5678/AutoTyper.git
```

### 2️⃣ Navigate to the Project

```bash
cd AutoTyper
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Script

```bash
python main.py
```
